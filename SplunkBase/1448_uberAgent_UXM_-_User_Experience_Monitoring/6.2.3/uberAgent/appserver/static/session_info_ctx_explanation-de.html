Erklärung der Daten in der Tabelle

User
Name des angemeldeten Benutzers.
Host
Name des Computers, auf dem die Sitzung lief.
ID
RDS-Sitzungskennung.
Logon time
Anmeldezeitpunkt.
Last seen
Zeitpunkt, an dem uberAgent das letzte Mal Daten über diese Sitzung lieferte
Session end
Zeitpunkt, an dem die Sitzung beendet wurde. Ist leer, wenn die Sitzung am Ende des Suchintervalls noch lief.
Protocol
Remoting-Protokoll, das zur Verbindung verwendet wurde
Last state
Letzter aufgezeichneter Zustand. Typischerweise active oder disconnected.
Client name(s)
Namen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client IP(s)
IP-Adressen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client platform(s)
Plattformen der Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Client version(s)
Versionen des Citrix Receiver auf den Clients, von denen aus eine Verbindung zur Sitzung hergestellt wurde.
Display specs
Anzeigemodus (horizontal x vertical x color depth) der Sitzung auf den Clients.
Pub. name
Name der veröffentlichten Anwendung, die verwendet wurde, um die Sitzung zu starten.
Pub. app(s)
Namen aller in der Sitzung verwendeten veröffentlichten Anwendungen.
Last app state
Letzter Citrix app state. Mögliche Werte: active, pre-launched, lingering, app not running.
Encryption
Verschlüsselung des Remoting-Protokolls.
