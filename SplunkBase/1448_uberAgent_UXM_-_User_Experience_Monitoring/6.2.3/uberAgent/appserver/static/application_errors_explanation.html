Explanation of the data in the table

Process name
Names of the process that crashed
Process version(s)
Version(s) of the process that crashed
App name(s)
Name of the application as determined by uberAgent's algorithms
#Crashes
Number of crashes for this process
User count
Number of affected users
Host count
Number of affected Hosts
Exception code(s)
Reason for the error
Faulting module(s)
Module where the error occured
Avg. process lifetime
Total lifetime of the process until the error
